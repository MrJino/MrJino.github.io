Android의 최신 UI 프레임워크인 Jetpack Compose를 배웁니다.

## 학습 목표

- Jetpack Compose가 무엇인지 이해합니다

- 첫 번째 Composable 함수를 작성합니다
- 기본 UI 컴포넌트를 사용합니다

## 1. Jetpack Compose란?

Jetpack Compose는 Android의 **선언형 UI 툴킷**입니다.

### 기존 View 시스템 vs Compose

**기존 방식 (XML + View)**:
```xml
<!-- XML 레이아웃 -->
<TextView
    android:id="@+id/textView"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:text="Hello" />
```

```kotlin
// Activity에서
val textView = findViewById<TextView>(R.id.textView)
textView.text = "Hello, World!"
```

**Compose 방식**:
```kotlin
@Composable
fun Greeting() {
    Text(text = "Hello, World!")
}
```

### Compose의 장점

- ✅ **적은 코드**: XML 불필요, 코드만으로 UI 구성
- ✅ **직관적**: UI 상태가 변하면 자동으로 업데이트
- ✅ **Kotlin 기반**: 타입 안정성과 null 안정성
- ✅ **상호운용성**: 기존 View와 함께 사용 가능

## 2. 프로젝트 설정

### Compose 프로젝트 생성

Android Studio에서:
1. **New Project**
2. **Empty Compose Activity** 선택
3. 프로젝트 설정 완료

### build.gradle 확인

```gradle
android {
    buildFeatures {
        compose true
    }
    composeOptions {
        kotlinCompilerExtensionVersion = "1.5.1"
    }
}

dependencies {
    // Compose BOM (Bill of Materials)
    implementation platform('androidx.compose:compose-bom:2023.10.01')

    implementation 'androidx.compose.ui:ui'
    implementation 'androidx.compose.material3:material3'
    implementation 'androidx.compose.ui:ui-tooling-preview'
    implementation 'androidx.activity:activity-compose:1.8.0'

    debugImplementation 'androidx.compose.ui:ui-tooling'
}
```

## 3. Composable 함수

Compose에서 UI는 `@Composable` 함수로 정의됩니다.

### 기본 구조

```kotlin
@Composable
fun MyScreen() {
    Text(text = "Hello, Compose!")
}
```

### MainActivity 설정

```kotlin
class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContent {
            MyAppTheme {
                Surface(
                    modifier = Modifier.fillMaxSize(),
                    color = MaterialTheme.colorScheme.background
                ) {
                    MyScreen()
                }
            }
        }
    }
}
```

fun TextExamples() {

## 6. Modifier

Modifier는 Composable의 크기, 모양, 동작을 변경합니다.

```kotlin
@Composable
fun ModifierExample() {
    Text(
        text = "Styled with Modifier",
        modifier = Modifier
            .fillMaxWidth()
            .padding(16.dp)
            .background(Color.LightGray)
            .border(2.dp, Color.Blue)
            .clickable { /* 클릭 이벤트 */ }
    )
}
```

### 주요 Modifier

- `size()`: 크기 설정
- `padding()`: 여백 설정
- `background()`: 배경색
- `fillMaxWidth()` / `fillMaxHeight()`: 최대 크기
- `weight()`: 가중치 (Row/Column에서)
- `clickable()`: 클릭 이벤트

## 7. 실전 예제

간단한 프로필 카드를 만들어봅시다:

```kotlin
@Composable
fun ProfileCard() {
    Card(
        modifier = Modifier
            .fillMaxWidth()
            .padding(16.dp),
        elevation = CardDefaults.cardElevation(defaultElevation = 4.dp)
    ) {
        Row(
            modifier = Modifier.padding(16.dp),
            verticalAlignment = Alignment.CenterVertically
        ) {
            // 프로필 이미지
            Image(
                painter = painterResource(id = R.drawable.profile),
                contentDescription = "Profile",
                modifier = Modifier
                    .size(64.dp)
                    .clip(CircleShape)
            )

            Spacer(modifier = Modifier.width(16.dp))

            // 이름과 직업
            Column {
                Text(
                    text = "John Doe",
                    fontSize = 20.sp,
                    fontWeight = FontWeight.Bold
                )
                Text(
                    text = "Android Developer",
                    fontSize = 14.sp,
                    color = Color.Gray
                )
            }
        }
    }
}
```
